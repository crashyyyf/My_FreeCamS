## Welcome

This repository contains **Cheat Engine Auto Assembler scripts** that provide **free camera control** in supported games.

These scripts are mainly created to explore **camera logic**, **memory structures**, and **reverse engineering techniques**.

---

## Features

- Full free camera movement (XYZ + rotation)
- Option to detach camera from player/entity
- Suitable for screenshots, video recording, and exploration

---

## Included Scripts

| Script Name        | Description                                   |
|--------------------|-----------------------------------------------|
| `full_freecam.asm` | Complete free camera implementation           |
| `hook_freecam.asm` | Demonstrates camera formulas and hook logic   |
